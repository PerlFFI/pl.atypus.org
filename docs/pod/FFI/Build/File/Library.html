NAME

FFI::Build::File::Library - Class to track object file in FFI::Build

VERSION

version 2.00

SYNOPSIS

 use FFI::Build;
 
 my $build = FFI::Build->new(source => 'src/*.c');
 # $lib is an instance of FFI::Build::File::Library
 my $lib = $build->build;

DESCRIPTION

This is a class to track a library generated by FFI::Build. This is returned by FFI::Build's build method. This class is a subclass of FFI::Build::File::Base. The most important method is probably path, which returns the path to the library which can be passed into FFI::Platypus for immediate use.

METHODS

path

 my $path = $lib->path;

Returns the path of the library.
