NAME

Alien::FFI - Build and make available libffi

VERSION

version 0.25

SYNOPSIS

In your Build.PL:

 use Module::Build;
 use Alien::FFI;
 my $builder = Module::Build->new(
   ...
   configure_requires => {
     'Alien::FFI' => '0',
     ...
   },
   extra_compiler_flags => Alien::FFI->cflags,
   extra_linker_flags   => Alien::FFI->libs,
   ...
 );
 
 $build->create_build_script;

In your Makefile.PL:

 use ExtUtils::MakeMaker;
 use Config;
 use Alien::FFI;
 
 WriteMakefile(
   ...
   CONFIGURE_REQUIRES => {
     'Alien::FFI' => '0',
   },
   CCFLAGS => Alien::FFI->cflags . " $Config{ccflags}",
   LIBS    => [ Alien::FFI->libs ],
   ...
 );

DESCRIPTION

This distribution installs libffi so that it can be used by other Perl distributions. If already installed for your operating system, and it can be found, this distribution will use the libffi that comes with your operating system, otherwise it will download it from the Internet, build and install it for you.
